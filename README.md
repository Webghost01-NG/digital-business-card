# Digital Business Card

A modern Digital Business Card built with React and Vite.  
This project presents personal information in a clean and interactive card format with animations, social media links, and theme switching.

## Features

- Dark and Light theme toggle
- Responsive layout (mobile and desktop)
- Profile image with glow animation
- Hover animations for buttons and icons
- Social media links
- Clean component-based React structure
- Modern glassmorphism UI design

## Technologies Used

- React
- Vite
- CSS3
- React Icons
